# 🚀 Aditya Ram — Developer Portfolio

A modern, responsive personal portfolio website built with **React**, **Vite**, and **Tailwind CSS** — showcasing my skills, projects, and experience as a Full Stack Developer.

🌐 **Live Demo:** [Portfolio](https://adityaram-portfolio.vercel.app/)

---

## ✨ Features

- **Responsive Design** — Fully optimized for desktop, tablet, and mobile devices
- **Modern UI** — Clean, professional interface styled with Tailwind CSS utility classes
- **Fast Performance** — Powered by Vite for lightning-fast builds and hot module replacement
- **Component-Based Architecture** — Modular React components for easy maintenance and scalability
- **Deployed on Vercel** — Continuous deployment with zero-config hosting

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Adityaram544/FUTURE_FS_01.git
   cd FUTURE_FS_01
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:5173
   ```

### Build for Production

```bash
npm run build
```

The optimized output will be in the `dist/` folder.

### Preview Production Build

```bash
npm run preview
```

## 🌐 Deployment

This project is deployed on **Vercel** with automatic deployments on every push to the `main` branch.

To deploy your own version:

1. Fork this repository
2. Connect it to your [Vercel account](https://vercel.com/)
3. Vercel will auto-detect the Vite configuration and deploy instantly
